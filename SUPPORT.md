# Obtenir de l'aide

Merci d'utiliser **KuT** ✂️ — voici par où passer selon votre besoin.

## 📚 Documentation

Avant tout, consultez la documentation :

- [`README.md`](README.md) — présentation, fonctionnalités et déploiement
- [`docs/sync-drive/`](docs/sync-drive/) — synchronisation Google Drive
- [`docs/multi-appareils/`](docs/multi-appareils/) — utiliser KuT sur plusieurs appareils
- [`docs/campagnes/`](docs/campagnes/) — campagnes email
- [`apps-script/README.md`](apps-script/README.md) — déploiement du script Apps Script
- Le **guide intégré** : bouton **« Revoir le guide »** dans **⚙️ Mon salon**

## 💬 Poser une question

Pour une question d'utilisation (« comment faire pour… ? »), ouvrez une
**[Discussion](https://github.com/J-Rbs91/ciseaux/discussions)**. C'est l'endroit
idéal pour échanger sans créer de ticket.

Vous pouvez aussi utiliser le bouton **« Faites-moi un retour »** en bas de la
page d'accueil de l'application.

## 🐛 Signaler un bug ou proposer une idée

Ouvrez une **[issue](https://github.com/J-Rbs91/ciseaux/issues/new/choose)** avec
le modèle adapté :

- **🐛 Signaler un bug** — décrivez les étapes, le comportement attendu, votre
  navigateur et appareil
- **✨ Proposer une fonctionnalité** — expliquez le besoin côté coiffeur

Avant d'ouvrir un ticket, vérifiez qu'un sujet
[existant](https://github.com/J-Rbs91/ciseaux/issues?q=is%3Aissue) ne couvre pas
déjà votre cas.

## 🔒 Faille de sécurité

**Ne signalez jamais une faille de sécurité dans une issue publique.** Suivez la
procédure décrite dans [`SECURITY.md`](SECURITY.md) (contact privé par email).
