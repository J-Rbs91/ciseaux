Campagnes email

KuT permet d'envoyer des offres par email à vos clients, depuis votre propre adresse Gmail, grâce au même script Apps Script que la synchro Drive. Aucun service tiers, aucune donnée hébergée ailleurs.

Comment ça marche ?

  1. Vos clients sont stockés dans votre Google Sheet (via la synchro).
  2. Vous rédigez un objet + un message dans la page Campagnes.
  3. Votre script envoie un email personnalisé à chaque client qui a accepté les offres.
  4. Chaque email part de votre Gmail et contient un lien de désinscription automatique.

Pré-requis

  • Avoir configuré la synchro Google Drive.
  • Avoir redéployé la dernière version du script (le code gère désormais l'envoi d'emails) et réautorisé l'accès — Google demandera l'autorisation d'envoyer des emails en votre nom.
  • Avoir coché « Accepte les offres » pour les clients concernés, puis synchronisé vers le Drive.
Quota Gmail
Un compte Gmail gratuit peut envoyer à 100 destinataires par jour (1 500 pour Google Workspace). Au-delà, les envois sont reportés au lendemain. La page Campagnes affiche votre quota restant.

Cibler les bons clients (segmentation)

Plutôt que d'écrire à toute votre base, choisissez un segment de destinataires dans la page Campagnes. Tous restent filtrés par l'opt-in (seuls les clients ayant accepté les offres sont contactés) :

  • Tous les clients opt-in — l'ensemble de vos clients consentants.
  • Récompense disponible — ceux qui ont atteint un palier de fidélité non encore utilisé.
  • À relancer — clients en sommeil, sans passage depuis un moment.
  • Clients fidèles — au-delà d'un nombre de passages que vous fixez.
  • Clients VIP — au-dessus d'un panier moyen que vous fixez.
  • Ont pris une prestation — ceux ayant déjà réservé une prestation donnée.

Personnalisation

Insérez ces variables dans l'objet ou le message, elles sont remplacées pour chaque client :

  • {prenom} — le prénom du client
  • {nom} — le nom complet
  • {offre} — le nom de l'offre liée (mails d'anniversaire)
  • {prestations} — la ou les prestations offertes (mails d'anniversaire)

Exemple : « Bonjour {prenom}, profitez de… ».

🎂 Mails d'anniversaire automatiques

Depuis la page Offres, cochez « Offre anniversaire », choisissez la prestation offerte et rédigez l'objet + le message. Tant que l'offre est active, le mail part automatiquement chaque année, le jour de l'anniversaire, à chaque client opt-in dont la date de naissance est renseignée — une seule fois par an et par client.

L'activation se fait une seule fois dans l'éditeur Apps Script (déclencheur quotidien). Voir la procédure dans apps-script/README.md.

Conformité RGPD
Seuls les clients ayant explicitement accepté les offres reçoivent vos emails, et chaque message contient un lien de désinscription qui met à jour automatiquement votre base. C'est la base du marketing par email responsable.

Astuces

  • Gardez vos messages courts (≈1500 caractères) — limite technique de l'envoi.
  • Cliquez « Vérifier les destinataires » avant d'envoyer pour connaître le nombre exact.
  • Ne fermez pas la page pendant l'envoi.
