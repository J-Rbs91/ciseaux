Synchronisation Google Drive

Cette méthode permet de retrouver vos données depuis n'importe quel appareil (tablette, téléphone, autre ordi) grâce à votre propre compte Google.

Comment ça marche ?

Vous déployez un petit script sur votre Google Drive. Il stocke vos données dans un fichier que vous seul contrôlez. 6-o n'héberge rien.

Étapes (5 min)

1
Ouvrir script.google.com et cliquer Nouveau projet.
2
Supprimer le code existant. Copier-coller le contenu du fichier apps-script/Code.gs disponible dans le dépôt GitHub du projet.
3
Cliquer Enregistrer (icône disquette).
4
Cliquer Déployer → Nouveau déploiement. Choisir le type Application web.
5
Paramètres :
• Exécuter en tant que : Moi
• Qui a accès : Tout le monde
6
Cliquer Déployer. Si un écran « application non vérifiée » s'affiche → Paramètres avancés → Continuer. C'est normal pour un script personnel.
7
Copier l'URL qui se termine par /exec.
8
Dans 6-o, ouvrir ⚙️ Mon salon, coller l'URL dans le champ Synchro Google Drive, puis enregistrer.
Compte Google Workspace (pro) ?
L'option « Tout le monde » peut ne pas être disponible. Solution : créer un compte Gmail personnel dédié (ex. monsalon.sync@gmail.com) et déployer le script depuis ce compte.

Vos données restent chez vous

Le script tourne sur votre compte Google. Il crée un dossier Hub_Facilities dans votre Drive avec vos données. 6-o n'y a pas accès.
